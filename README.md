# Monday Advertising Website

A modern, responsive website for Monday Advertising, a paid ads marketing agency.

## Features

- Responsive design that works on all devices
- Modern and clean UI with smooth animations
- Interactive elements including testimonial slider
- Contact form with validation
- Optimized for performance and SEO

## Structure

- `index.html` - Main HTML file
- `css/styles.css` - CSS styles
- `js/main.js` - JavaScript functionality
- `images/` - Directory for website images

## Getting Started

To view the website locally:

1. Clone or download this repository
2. Open the `index.html` file in your browser
3. Alternatively, you can use a local server to serve the files

## Image Placeholders

The website uses placeholder images that should be replaced with actual company images:

- Hero image
- About section team image
- Case study images
- Testimonial profile images

- HTML5
